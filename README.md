# Cradlewell 🌿    

https://cradelwell.netlify.app/ -- the link to th ewebsite

> Nurse-led postnatal & newborn care, delivered home — Bangalore

A fully responsive single-page marketing website for **Cradlewell**, a healthcare service providing expert nurse-led care to mothers and newborns during the critical first 40 days after birth.

---

## ✨ Features

- Animated heart-wave loading screen
- Smooth slide-left + fade page transition
- Scroll-triggered fade-up animations (IntersectionObserver)
- Responsive layout — mobile, tablet & desktop
- Services, pricing plans, team, FAQ, and testimonials sections
- WhatsApp CTA integration
- Two-image floating animation in "Why It Matters" section
- Team section with real founder photos

## 🚀 Getting Started

No build step needed — it's a single HTML file.

1. Clone the repo:
   ```bash
   git clone https://github.com/MSreeGaneshNaik/cradlewell.git
   ```
2. Open `cradlewell.html` in your browser — done.

## 🛠 Tech Stack

- Vanilla HTML / CSS / JavaScript
- Google Fonts — Playfair Display + DM Sans
- CSS custom properties for theming
- IntersectionObserver API for scroll animations
- Canvas-free SVG heart-wave animation

## 📍 About Cradlewell

Cradlewell connects new families with verified, background-checked nurses for in-home postnatal care across Bangalore — covering newborn health, lactation support, mother recovery, and 24/7 guidance.
